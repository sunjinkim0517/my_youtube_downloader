# 한글 유튜브 다운로더 (YouTube Downloader - Korean Edition)

오픈소스 미디어 다운로더인 **Hitomi-Downloader**의 디자인과 핵심 기능을 벤치마킹하여, YouTube 플랫폼에 최적화된 고성능 한국어 데스크톱 다운로더 프로그램입니다.

---

## 주요 기능 (Features)
1. **현대적인 다크 테마 GUI:** 사용자의 눈이 편안한 어두운 색상의 인터페이스와 진행 표시바 제공.
2. **비동기 멀티스레딩:** 다운로드 작업이 백그라운드에서 실행되므로, 대량의 다운로드 중에도 프로그램이 멈추지 않고 매끄럽게 동작합니다.
3. **다운로드 큐(Queue) 시스템:** 동시 다운로드 개수를 설정하여 한 번에 실행되는 다운로드 개수를 관리할 수 있습니다.
4. **포맷 및 화질 제어:** 
   - **비디오 (MP4):** 최고 화질, 1080p, 720p, 480p, 360p 선택 가능.
   - **오디오 (MP3):** 오디오 트랙만 추출하여 MP3 형식으로 저장 (320kbps, 256kbps, 192kbps, 128kbps 선택 가능).
5. **재생목록(Playlist) 지원:** 유튜브 재생목록 URL을 복사해 넣으면 일괄 분석하여 목록 안의 동영상을 한 번에 추가할 수 있습니다.
6. **클립보드 감지 기능:** 웹브라우저에서 유튜브 링크를 복사하면 프로그램이 자동으로 감지하여 주소창에 입력해 줍니다.
7. **FFmpeg 내장 지원:** `imageio-ffmpeg` 라이브러리를 내장하여, 별도로 FFmpeg를 설치할 필요 없이 1080p 고화질 병합 및 MP3 변환이 가능합니다.

---

## 실행 방법 (How to Run)

### 1. 필수 라이브러리 설치
본 프로그램은 Python 3.10 이상 환경을 권장합니다. 아래 명령어로 필요한 패키지들을 설치하세요.

```bash
pip install PySide6 yt-dlp imageio-ffmpeg
```

*참고: 이미 설치 환경에 맞게 라이브러리 의존성이 사전에 완료되었습니다.*

### 2. 프로그램 실행
터미널 또는 명령 프롬프트(CMD)에서 아래 명령어를 입력하여 프로그램을 실행합니다.

```bash
python app.py
```

---

## 단일 실행 파일(.exe)로 빌드하기 (Build to Executable)

윈도우 환경에서 파이썬이 설치되지 않은 사용자도 사용할 수 있도록 하나의 실행 파일(`.exe`)로 만들 수 있습니다.

1. **PyInstaller 설치:**
   ```bash
   pip install pyinstaller
   ```

2. **빌드 명령어 실행:**
   ```bash
   pyinstaller --noconsole --onefile --name="YouTube_Downloader" app.py
   ```
   * `--noconsole`: 프로그램을 실행할 때 터미널 검은 창이 뜨지 않도록 설정합니다.
   * `--onefile`: 모든 의존 파일 및 아이콘을 `.exe` 단일 파일로 묶어줍니다.

3. 빌드가 완료되면 `dist` 폴더 안에 `YouTube_Downloader.exe` 파일이 생성됩니다.
